# 退休金计算器技术规格说明（v1）

## 1. 目标

交付一个可运行的纯前端退休金计算器，面向城镇职工养老保险（含灵活就业），输出退休当年月养老金估算结果。

## 2. 范围与边界

- **包含**：基础养老金、个人账户养老金、过渡性养老金（手动输入）。
- **不包含**：城乡居民养老保险、按省份自动计算过渡性养老金。
- **输出口径**：退休当年月金额（元/月），统一保留两位小数。

## 3. 核心公式

- 总公式：`total = basic + personal + transitional`
- 基础养老金：`basic = P * (1 + i) / 2 * n * 0.01`
- 个人账户养老金：`personal = accountBalance / divisorByAge(retireAge)`

## 4. 输入模型

```ts
type RetireAge = 40 | 45 | 50 | 55 | 60 | 65 | 70;
type AccountMode = "DIRECT_BALANCE" | "ESTIMATE";
```

关键字段：

- `retireAge`：退休年龄（决定计发月数）
- `P`：退休上年度当地在岗职工月平均工资
- `i`：平均缴费指数
- `n`：累计缴费年限
- `accountMode`：个人账户计算模式
- `personalRate`：个人费率（默认 0.08，允许作为模拟参数调整）
- `transitional`：过渡性养老金（结合开关生效）

## 5. 校验规则

- `P > 0`
- `n > 0`
- `i >= 0`
- 退休年龄必须出现在计发月数表中
- 模式 A：`accountBalance` 必填且 `>= 0`
- 模式 B：`monthlyBase > 0`、`years > 0` 且为整数、`personalRate > 0`、`annualInterestRate >= 0`
- `n < 15` 时给出提示但不阻断计算

## 6. 架构设计

```text
src/
  pages/PensionCalculator.vue      # 页面编排与状态联动
  components/InputForm.vue         # 参数输入
  components/ResultPanel.vue       # 计算结果与提示
  components/FormulaPanel.vue      # 公式与计发月数说明
  core/pension.ts                  # 纯计算函数
  core/divisor-table.ts            # 计发月数常量表
  core/validate.ts                 # 纯校验逻辑
  types/pension.ts                 # 类型定义
  utils/storage.ts                 # localStorage 持久化
```

## 7. 持久化与交互

- 页面初始化从 `localStorage` 读取输入值（无值则使用默认值）。
- 任一输入变化后自动写回 `localStorage`。
- 实时计算，不设置手动“计算”按钮。

## 8. 测试策略

- 使用 Vitest 对核心公式进行单元测试：
  - 直接输入模式计算正确性
  - 估算模式账户储存额计算正确性
  - 边界提示与校验错误覆盖
